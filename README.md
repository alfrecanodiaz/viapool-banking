## Description

[Viapool](https://viapool.com/) banking coding task.

## Requirements
- Node v12 or similar.
- NPM v6 or similar.
- PostgreSQL installed and running.
- Env file configured with correct values (take a look at the sample).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```
